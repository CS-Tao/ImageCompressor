# ImageCompressor
一个图片压缩工具

## 使用方法

### 克隆本仓库

```Bash
git clone https://github.com/CS-Tao/ImageCompressor.git
```

### 运行脚本

1. 打开命令行终端，cd进入ImageCompressor.py所在文件夹(Linux系统直接在对应文件夹中打开命令终端)
2. 执行以下命令：
    ```Bash
    python ImageCompressor.py -i [源图片文件夹] -o [压缩图片的输出路径] -s [压缩尺度，可取1、2、3、4，值越大，压缩越明显]
    ```
